Day #5 : CSS :has()

This pseudo-class search for any tag, class or selectors relative to current selector. If any found then it styles the current selector.

:has({selector}){
      ...
}

For example, give red color to all h1 elements if it has an item with class 'red';

h1:has(.red){
      color:red;
}
<h1>H1 element with <span class="red">red class</span></h1>

Second example, give blue color to all h1 elements if any h2 element comes after h1;

h1:has(+h2){
      color:blue;
}
<h1>H1 element with H2 Element</h1>
<h2>H2 as sub title</h2>

Day #4 : CSS Rotate

Css de rotate bir elemanı saat yönünde veya tersinde belirli bir açıyla döndürmek için kullanılır. Özellikle transform ile birlikte kullanıldığında şık animasyonlar elde etmek mümkündür. alışveriş sitelerinde kart bilgilerinizi girdiğiniz ekrandaki dönen kart animasyonuna aşinasınızdır. Bir örneğini aşağıda bulabilirsiniz.

.item {
      rotate: 45deg;
}
.item:hover {
      transform: rotate(45deg);
}

Day #1 : clamp() fonksiyonu

clamp() fonksiyonu bir minimum ve maksimum aralıkta responsive olarak içerik boyutuna göre değer atamamızı sağlar. Tarayıcı eğer ortadaki parametre belirtilen aralıkta ise onu kullanır yoksa alt ve üst limitlere göre hareket eder.

.item {
      width: clamp(min_width, preferred, max_width);
      width: clamp(150px, 50%, 400px);
      font-size: clamp(2rem, 4vw, 4rem);
}


Output

.clamp-test {
      width: clamp(250px, 30%, 600px);
      padding: clamp(1rem, 2vw, 2rem);
      background: #00ffd0;
}
