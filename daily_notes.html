Day #8 CSS : Border Animation

Today we will animate the html borders using only css.

First create our html content;


            <div class="border-animate">Loading.. Please wait..</div>

Then, add the css rules. We will use conic-gradient property to animate the border for pseudo ::after and ::before elements;


            @property --angle{
                syntax: '';
                initial-value: 0deg;
                inherits: false;
            }

            .border-animate{margin: 0 auto; padding: 2em 0; text-align: center;width: 300px;
                background-color: #fff;border-radius: 10px;position: relative;}

            .border-animate::after, .border-animate::before{content: '';position: absolute;height: 100%;width: 100%;
                background: conic-gradient(from var(--angle), #ff4545, #00ff99, #006aff, #ff0095, #ff4545); 
                border-radius: 10px;top:50%;left:50%;translate:-50% -50%;z-index: -1;padding: 6px;
                animation: 3s spin linear infinite;}
                
            .border-animate::before{
                filter: blur(1.5rem);
                opacity: 0.5;
            }

            @keyframes spin {
                from{--angle:0deg;}
                to{--angle:360deg;}
            }

Day #8 JS: Nummber Counter

You may have come across sections on corporate websites titled 'Our Services in Numbers.' Today, let's see how we can create one ourselves!

First create the html content;


            <div class="numeric-counter-container">
                <div class="numeric-counter"><i class="fas fa-users"></i><span data-max="34100">0</span> User</div>        
                <div class="numeric-counter"><i class="fas fa-book"></i><span data-max="161346">0</span> Session</div>                
                <div class="numeric-counter"><i class="fas fa-turkish-lira"></i><span data-max="55713">0</span> Payment</div>    
            </div>

Then, create the css content;


            .numeric-counter-container{display: flex;justify-content: space-around;font-size: 20px;text-align: center;font-weight: bold;}            
            .numeric-counter-container span{display: block;font-size: 30px;color: #2678bf;}

Then, create the javascript content;


            $(document).ready(function () {
                $(".numeric-counter").each(function () {
                    var max = $(this).find("span").attr("data-max");                    
                    $(this).children("span").animate({
                        Counter: max
                    }, {
                        duration: 4e3,
                        easing: "swing",
                        step: function (now) {
                            $(this).text(Math.ceil(now));
                        }                
                    });    
                })      
            });

Output

0 User
0 Session
0 Payment

Day #6 : CSS only Tooltip

Today we will create a simple css only tooltip by using ::before pseudo-class. To display tooltip content we will use the content property.

First create the html content;

<div>This text contains a tooltip example. <span class="css-tooltip">HOVER ME!!!</span></div>

Then define CSS rules;

.css-tooltip{position: relative;cursor: pointer;font-weight: 600;color:red;}
.css-tooltip::before{content: 'A nice tooltip'; display: none;padding: 3px 7px; border-radius: 6px;color: #2678bf;
      position: absolute;top: -27px;font-size: 13px; width: max-content;border: 1px solid #2678bf;}
.css-tooltip:hover::before{display:inline-block;}


Output

This text contains a tooltip example. HOVER ME!!!

Day #5 : CSS :has()

This pseudo-class search for any tag, class or selectors relative to current selector. If any found then it styles the current selector.

:has({selector}){
      ...
}

For example, give red color to all h1 elements if it has an item with class 'red';

h1:has(.red){
      color:red;
}
<h1>H1 element with <span class="red">red class</span></h1>

Second example, give blue color to all h1 elements if any h2 element comes after h1;

h1:has(+h2){
      color:blue;
}
<h1>H1 element with H2 Element</h1>
<h2>H2 as sub title</h2>

Day #1 : clamp() fonksiyonu

clamp() fonksiyonu bir minimum ve maksimum aralıkta responsive olarak içerik boyutuna göre değer atamamızı sağlar. Tarayıcı eğer ortadaki parametre belirtilen aralıkta ise onu kullanır yoksa alt ve üst limitlere göre hareket eder.

.item {
      width: clamp(min_width, preferred, max_width);
      width: clamp(150px, 50%, 400px);
      font-size: clamp(2rem, 4vw, 4rem);
}


Output

.clamp-test {
      width: clamp(250px, 30%, 600px);
      padding: clamp(1rem, 2vw, 2rem);
      background: #00ffd0;
}
